# Solnero Frontend

Frontend application for Solnero - Private Solana Transactions.

## Features

- Wallet creation and management
- Private transaction sending using zero-knowledge proofs
- Transaction history (user-only visibility)
- Secure key management
- Modern UI with Tailwind CSS and shadcn/ui

## Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
